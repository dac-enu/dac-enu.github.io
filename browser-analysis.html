Web Browser Analysis

Below you will see a button that will exectute JavaScript code to give you a sample of some of the data points that your web browser can report to a website even without you knowing, unless you are already aware of it being a possibility. These data points as well as others can be used to create a unique digital fingerprint and with common trackers deployed on various websites from the likes of Google, Cloudflare, Meta, and more, this will allow them to see what websites and its pages you visit.
